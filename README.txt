F# Codebeispiele Readme
***************************************************************

Um die F# Codebeispiele auszuführen wird "Microsoft Visual Studio" enpfohlen.

Um die Codebeispiele zu öffnen, soll die Datei "FSharpIntro.sln" im Ordner "FSharpIntro" geöffnet werden. 
Je nach dem ob F# bereits installiert ist, wird man von Visual Studio gebeten, F# zu installierern. Dies macht Vistual Studio allerdings autonom.

Nach der Installation ist können die Codebeispiele bereits kompilliert und ausgeführt werden.
